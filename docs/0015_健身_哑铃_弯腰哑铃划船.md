# 弯腰哑铃划船
- 抓住两个哑铃，双脚分开与肩同宽站立，手掌相对，抓住哑铃
- 保持背部挺直的同时，稍微弯曲膝盖并向前倾斜，使脊柱与地板成 45-90 度
